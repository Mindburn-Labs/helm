# Rust Client Example

Shows HELM integration with the Rust SDK.

## Prerequisites

- HELM running at `http://localhost:8080` (`docker compose up -d`)
- Rust 1.75+

## Run

```bash
cargo run
```

## Expected Output

```
Chat Completions: Denied: DENY_TOOL_NOT_FOUND
Evidence: Exported 1234 bytes
Conformance: PASS (12 gates)
```
